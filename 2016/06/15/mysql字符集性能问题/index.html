mysql 相关操作

忘记密码

  1. 在 my.ini 文件的 mysqld 下面添加skip-grant-tables，保存退出。
  2. 重启 mysql, service mysql restart
  3. 直接输入 mysql 就可以登录成功

  4. 改密码

然后，使用新密码登录即可。如果不成功，重启一下mysql服务。
